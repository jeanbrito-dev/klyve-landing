# Klyve Landing Page

Landing page oficial do **Klyve**, um sistema financeiro visual para qualquer negócio.

O site apresenta a proposta da plataforma com foco em organização, clareza e controle financeiro de forma simples e intuitiva. A comunicação é objetiva e voltada para destacar a visualização das finanças como diferencial estratégico.

---

## 🚀 Sobre o projeto

A landing page foi desenvolvida utilizando:

- HTML  
- CSS  
- JavaScript  

O design segue uma identidade moderna em tons de âmbar combinados com neutros claros, transmitindo energia, tecnologia e confiança — pilares essenciais para um SaaS financeiro.

---

## 🎯 Objetivo

Apresentar o Klyve como uma solução prática de organização financeira e direcionar o usuário para contato direto via:

- WhatsApp (canal principal)  
- Instagram (@klyve.oficial)  

---

## 💡 Proposta

Transformar números soltos em visão clara, facilitando decisões e trazendo mais controle sobre o financeiro do negócio.
